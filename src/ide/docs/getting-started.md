# Getting Started

Welcome to the documentation. This is a sample page to get you started.

## Overview

This extension provides a scripting IDE for Sitecore XM Cloud, allowing you to write and execute scripts directly within the Sitecore Marketplace.

## Quick Start

1. Install the extension from the Sitecore Marketplace
2. Open the IDE from your dashboard
3. Start writing scripts
